The Fate Fantasy SRD is a free and open resource that you can use in your own campaigns, books and materials. It is inspired by the Fate SRD.

This web site is only possible thanks to the generosity of Evil Hat Productions, the publishers of Fate Core.

Licensing

We want this material to be easily and freely accesible to everyone.

So the material on this web site is free or open content. This means you can borrow, steal or re-write anything you see here as long as you follow the terms of the Creative Commons Attribution License.

To use the content on the web site follow the official instructions for licensing the Fate Core RPG. When copying the attribution notice don't forget to include our copyright notice, as follows:

© 2016 Ricardo Gladwell. The Fate Fantasy SRD by Ricardo Gladwell is licensed under a Creative Commons Attribution 4.0 International License.

Source Code

We're looking to continually improve this web site and we encourage you to send updates if you spot typos, mistakes or if you have ideas yourself.

The complete source code to build web site is can be found on Github so go there and send us some code.

How we made this site

This site is made from blood, sweat and tears. But credit where credit is due:

  • Runs on the Javascript-based Node.js/NPM stack
  • Built with Grunt
  • Styled using Bootstrap and libsass
  • Uses the Quattrocento and Fanwood Text fonts
